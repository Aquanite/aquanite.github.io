# CHance programming language
Welcome to the CHance programming language, you can view the source code [here](https://github.com/Aquanite/CHance),
view the releases [here](https://github.com/Aquanite/CHance/releases), and view the documentation on this website!

## Installation guide
To install the CHanceC Compiler, you will need to follow the below steps.

1. Clone the repositories
```bash
git clone https://github.com/Aquanite/CEnhanced.git
git clone https://github.com/Aquanite/ChanceCode.git
git clone https://github.com/Aquanite/CHS.git
git clone https://github.com/Aquanite/CLD.git
```

2. Build all of the ChanceC related tools

Linux:
```bash
cd CEnhanced
./install.sh
```

Windows:
```batch
cd CEnhanced
.\install.bat
```
